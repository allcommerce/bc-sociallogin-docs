Script Installation¶
After installing the application, you need to install the script to display login buttons on your website.

Method 1: Automatic (Recommended)¶
Papa Social Login supports automatic script installation:

Open Papa Social Login dashboard
Go to "Installation" tab
Click "Auto Install Script"
Choose appropriate option:
Auto Install Options

Installation options:

Install on login page only: Only install on login page
Install on checkout page only: Only install on checkout page
Install on all pages: Install on all login/checkout pages
Custom installation: Customize installation location
Method 2: Manual¶
If auto install doesn't work or you want detailed control:

Step 1: Get Script Code¶
In Papa Social Login dashboard
Go to "Installation" → "Manual Installation"
Copy the provided script:

<script>
  (function() {
    var script = document.createElement('script');
    script.src = 'https://app.papasociallogin.com/scripts/social-login.js';
    script.setAttribute('data-store-hash', 'YOUR_STORE_HASH');
    script.setAttribute('data-channel-id', 'YOUR_CHANNEL_ID');
    document.head.appendChild(script);
  })();
</script>
Step 2: Insert Script into Theme¶
Method 1: Template Editing (Recommended)

In BigCommerce Admin, go to Storefront → Script Manager
Click "Create a Script"
Fill in the information:
Name: Papa Social Login
Description: Social login buttons
Location: Footer
Select pages: Login page, Checkout page
Script category: Essential
Script type: Script
Script Manager

Paste the script code into Script contents section
Click "Save"
Method 2: Direct Theme Editing

Warning

This method requires HTML/CSS knowledge. Always backup your theme before editing.

Go to Storefront → My Themes
Click "Advanced" → "Edit Theme Files"
Find the file templates/pages/auth/login.html
Add the script at the end of the file, before the </body> tag
Save changes
